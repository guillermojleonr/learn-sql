# Learn-sql
Proyecto para aprender y practicar SQL puro, no siempre será el mejor proyecto para aprender la sintaxis específica de Datawarehouses, Lakehouses, bases de datos y otros sistemas SQL-Like, las miles de features que tienen estos sistemas, para ello lo mejor sería crear un proyecto independiente para cada una de esas plataformas.

Aquí al aprender y practicar SQL puro mantenemos todo simple, mockeando data, usando bases de datos sencillas, emulando arquitecturas de datos que no necesariamente serán production grade. En ese sentido la idea de este proyecto es aprender y tener sólidas bases para escribir SQL con soltura, aprender conceptos, técnicas y patrones comunes

## Estructura

En sample_projects tenemos los distintos proyectos para practicar y fuera todo tipo de carpetas relacionadas con teoría.
